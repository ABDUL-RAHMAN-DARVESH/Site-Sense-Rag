# 🔍 SiteSence RAG - Website Q&A using Groq + FastAPI

**SiteSence RAG** is a Retrieval-Augmented Generation (RAG) system that takes a website URL, scrapes its content, and intelligently answers questions using Groq's blazing-fast LLaMA 3 model (`llama3-8b-8192`). This project combines FastAPI, FAISS, SentenceTransformers, and LangChain to deliver real-time, context-aware answers.

---

## 🚀 Features

- 🌐 Scrape any publicly available website
- 🧩 Smart text chunking with overlap
- 🤖 Sentence-level embeddings via `all-MiniLM-L6-v2`
- ⚡ Fast FAISS vector similarity search
- 🧠 RAG-based Q&A using Groq-hosted LLaMA 3
- 🛠️ Modular, beginner-friendly codebase
